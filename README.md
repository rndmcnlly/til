## How this works

Each TIL is a plain markdown file at `<topic>/<slug>.md`. On push to main, a GitHub Actions workflow runs `build.py` (a single-file site generator using [mistune](https://mistune.readthedocs.io/) and [`uv run --script`](https://docs.astral.sh/uv/guides/scripts/)), which renders all `*/*.md` files into static HTML and deploys to GitHub Pages.

Inspired by [simonw/til](https://github.com/simonw/til).
